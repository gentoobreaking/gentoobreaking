Tools for Scanning Source Code for Vulnerabilities

發表於 2012-09-11 更新於 2026-06-18 分類於 資安技術

商業版本

兩套比較有名的商業版本（都有試用版可下載，完整測試就要使用 crack 版本囉~）

  • IBM AppScan — IBM 推出的網站安全掃描工具，支援多種漏洞檢測
  • HP WebInspect — HP 的 Web 應用程式安全測試工具

延伸閱讀：當初將 IBM AppScan 做逆向工程後，改寫的另一套商業版本 Veracode
