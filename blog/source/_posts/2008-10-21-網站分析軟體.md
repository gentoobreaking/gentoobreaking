<h2>網頁流量統計工具比較：AWStats vs Webalizer</h2>

<p>與 長官聊到網頁分析統計軟體，主要需求是統計<code style="background:#f3f4f6;padding:2px 6px;border-radius:4px;color:#dc2626;">user download file 的 bandwidth</code>，希望能依目錄區分下載流量、列出被下載的檔案清單及流量報表。</p>

<hr/>

<h3>功能特色</h3>

<ul style="color:#374151;line-height:2;">
  <li>支援 Web、FTP、Mail、Squid、Streaming 多種 Log 格式</li>
  <li><strong>即時更新</strong>（可關閉即時功能僅保留靜態報表）</li>
  <li>可統計下載檔案、錯誤頁面、訪客来源、瀏覽器、作業系統等</li>
  <li>可依目錄或副檔名篩選，適合<strong>下載目錄流量分析</strong>需求</li>
  <li>輸出 HTML 報表，瀏覽器即可檢視</li>
</ul>

<hr/>

<h2>2. Webalizer</h2>

<div style="background:#eff6ff;border-left:4px solid #3b82f6;padding:12px 16px;margin:16px 0;border-radius:0 8px 8px 0;">
  <p style="margin:0;color:#1e40af;"><strong>💡</strong> <i>The Webalizer</i> is a fast, free web server log file analysis program. It produces highly detailed, easily configurable usage reports in HTML format, for viewing with a standard web browser.</p>
</div>

<p><strong>官方網站：</strong><a href="http://www.webalizer.org/" target="_blank" style="color:#1d4ed8;text-decoration:underline;">http://www.webalizer.org/</a></p>

<p><strong>台灣 ADJ 網路 Demo：</strong><a href="http://www.adj.idv.tw/usage/usage_200810.html#TOPAGENTS" target="_blank" style="color:#1d4ed8;text-decoration:underline;">http://www.adj.idv.tw/usage/usage_200810.html</a></p>

<h3>功能特色</h3>

<ul style="color:#374151;line-height:2;">
  <li>速度快、系統資源佔用低，適合大量 Log 檔案</li>
  <li>輸出 HTML 格式報表，含長條圖、線圖等視覺化圖表</li>
  <li>支援 ISP 匿名化（Hideous/hide IP）功能</li>
  <li>可依國家、地區、URL、Referer 等維度分析</li>
  <li>Unix / Linux 平台為主，也有 Windows 版</li>
</ul>
