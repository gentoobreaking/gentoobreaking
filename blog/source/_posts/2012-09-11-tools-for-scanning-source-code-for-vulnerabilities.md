## 商業版本

> 兩套比較有名的商業版本（都有試用版可下載，完整測試就要使用 crack 版本囉~）

- **[IBM AppScan](http://www-01.ibm.com/software/tw/promotion/rational/cases.html)** — IBM 推出的網站安全掃描工具，支援多種漏洞檢測
- **[HP WebInspect](https://download.hpsmartupdate.com/webinspect/)** — HP 的 Web 應用程式安全測試工具

> 延伸閱讀：當初將 IBM AppScan 做逆向工程後，改寫的另一套商業版本 **[Veracode](http://www.veracide.com/security/rational-appscan-alternative/)**

## 免費版本

> Open source tools for scanning source code for vulnerabilities?

### OWASP 官方推薦工具列表

- [OWASP Source Code Analysis Tools](https://www.owasp.org/index.php/Source_Code_Analysis_Tools) — OWASP 官方整理的工具清單

### OWASP 旗下三個開源專案

| 專案 | 說明 |
|------|------|
| [OWASP Orizon Project](https://www.owasp.org/index.php/Category:OWASP_Orizon_Project) | 程式碼安全分析框架 |
| [OWASP LAPSE Project](https://www.owasp.org/index.php/OWASP_LAPSE_Project) | Java 應用程式弱點掃描工具 |
| [OWASP O2 Platform](https://www.owasp.org/index.php/OWASP_O2_Platform) | 平台化的安全測試工具組 |

### 其他推薦工具
